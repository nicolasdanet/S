
# Spaghettis

    Yet another fork of Pure Data < http://msp.ucsd.edu/ >.

Platforms:

    Tested on Ubuntu 20.04, macOS 10.15, Manjaro and Raspbian Buster (RPI4).
        
Dependencies:

    On Debian-based following packages are required:
    
        - libasound2-dev
        - libjack-jackd2-dev
    
    On Arch-based following packages are required:
    
        - jack

Run:

    Build with the "Spaghettis/tests/build.sh" script.
    Then load a patch (e.g. "./spaghettis stuff/T.floyd.pdpatch stuff/preferences.txt").

Residual:

    ~/Library/Application Support/Spaghettis/
    ~/.config/spaghettis/
